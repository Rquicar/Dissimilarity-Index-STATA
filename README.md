# Dissimilarity-Index-STATA
STATA script to generate an Excel file with the Dissimilarity Index (by pairs and average), broken down by country and ISSP surveys. EGP 4 classes.
This script is made for a very specific task. Create an excel file with de ID and average ID of every country of the ISSP survey of social inequality by year using the EGP Schema with four classes (I+II, IIIab+IVab+V+VI, VIIa, VIIb+IVc). 

For more information about this classification:

Solís, P., & Boado, M. (2016). Y sin embargo se mueve. Estratificación social y movilidad intergeneracional de clase en América Latina. México: El Colegio de México y Centro de Estudios Espinosa Yglesias.

Fachelli, S., Marqués-Perales, I., Boado, M., Solís, P. (2021). Social Mobility from a Comparative Perspective Between Europe and Latin America. In: López-Roldán, P., Fachelli, S. (eds) Towards a Comparative Analysis of Social Inequalities between Europe and Latin America. Springer, Cham. https://doi.org/10.1007/978-3-030-48442-2_7
