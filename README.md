# Dissimilarity-Index-STATA
STATA script to generate an Excel file with the Dissimilarity Index (by pairs and average)

Part of the elaboration of this files has been carried out in the context of the INCASI2 project that has received funding from the European Union’s Horizon Europe research and innovation programme under the Marie Sklodowska-Curie grant agreement No 101130456 (https://incasi.uab.es).
